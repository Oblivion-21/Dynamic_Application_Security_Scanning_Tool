# Dynamic_Application_Security_Scanning_Tool
A tool to automatically run groups of security tests against website


# GUI

## Setup

```
cd gui
npm install
```

## Start

```
npm start
```

# Backend

## Setup

```
cd app
pip install pipenv
pipenv --python 3.10
pipenv --python 3.9
pipenv install
```

## Start

```
cd app
pipenv run python API.py
```
